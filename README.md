# Clima Tradicional y POO - Semana 03

**Autor:** Jenifer Fernanda Vaca Escobar  
**Asignatura:** Programación Orientada a Objetos  
**Tema:** Promedio semanal del clima en Python

Este repositorio contiene la solución a una tarea de programación en Python, implementada en dos versiones:

- `clima.py`: usando programación tradicional (estructurada)
- `clima_poo.py`: usando programación orientada a objetos (POO)

---

## Descripción de la tarea

Se solicitó un programa que permita al usuario ingresar las temperaturas diarias de una semana y calcule el promedio semanal.

El objetivo fue practicar ambos enfoques de programación: tradicional y orientado a objetos.

---

## Comparación entre Programación Tradicional y POO

| Característica             | Programación Tradicional                   | Programación Orientada a Objetos (POO)       |
|----------------------------|--------------------------------------------|----------------------------------------------|
| Estructura del código      | Uso de funciones sueltas                   | Uso de clases y métodos                      |
| Organización               | Menos estructurada                         | Más organizada y modular                     |
| Reutilización              | Limitada                                   | Alta, gracias a clases reutilizables         |
| Escalabilidad              | Menor                                      | Mayor, ideal para programas complejos        |
| Encapsulamiento            | No                                          | Sí, mediante clases                          |
| Ideal para                 | Tareas simples o scripts pequeños          | Aplicaciones grandes y mantenibles           |
